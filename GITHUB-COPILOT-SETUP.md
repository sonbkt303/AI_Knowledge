# Hướng dẫn nhanh GitHub Copilot

Mục tiêu: bắt đầu nhanh với Copilot cho dự án, ngắn gọn và dễ áp dụng.

Yêu cầu:
- VS Code (hoặc VS Code Insiders).
- Tài khoản GitHub (và Copilot nếu cần).

Các bước nhanh:
- Bật gợi ý nội tuyến (inline suggestions) để nhận gợi ý ngay khi gõ.
- Cài extension "GitHub Copilot" từ Marketplace.
- Đăng nhập GitHub trong VS Code nếu được yêu cầu.

Mẹo ngắn:
- Viết comment mô tả mục tiêu trước khi code để Copilot hiểu ý định.
- Luôn review code do Copilot sinh — kiểm tra bảo mật và logic.
- Dùng cấu hình workspace nếu muốn trải nghiệm đồng nhất cho team.

Quy tắc an toàn:
- Không lưu secrets trong mã nguồn.
- Thiết lập quy trình review khi chấp nhận mã gợi ý từ AI.

Khắc phục cơ bản:
- Reload cửa sổ VS Code nếu không thấy gợi ý.
- Kiểm tra Output → chọn `GitHub Copilot` để xem log.

Muốn thêm: tôi có thể tạo file mẫu ` .vscode/settings.json` hoặc thêm đoạn vào `CONTRIBUTING.md`.
